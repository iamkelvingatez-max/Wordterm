About WordTerm

WordTerm v1.0

A unique terminal emulator disguised as Microsoft Word, combining professional aesthetics with powerful Kali Linux capabilities.

Features:
• Full bash terminal with PTY support
• Smart autocomplete with command/file suggestions
• Auto-cd when typing directory paths
• Integrated Kali tool launcher
• Copy/paste with Ctrl+Shift+C/V
• Context menu support
• Multiple themes

Built with Electron + xterm.js + node-pty
